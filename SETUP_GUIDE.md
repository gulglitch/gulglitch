# 🎨 GitHub Profile Setup Guide

This guide will help you set up your awesome GitHub profile README!

## 📋 Step-by-Step Instructions

### 1. Create Your Profile Repository

1. Go to GitHub and create a **new repository**
2. Name it **exactly** as your GitHub username (e.g., if your username is `johndoe`, name it `johndoe`)
3. Make it **Public**
4. Check "Add a README file"
5. Click "Create repository"

### 2. Customize Your README

Replace the following placeholders in `README.md`:

#### **Replace `YOUR_GITHUB_USERNAME`** (appears multiple times)
- Find all instances of `YOUR_GITHUB_USERNAME`
- Replace with your actual GitHub username

#### **Update Personal Information**
- `[Your Name]` → Your real name
- `your.email@example.com` → Your email address
- `https://yourwebsite.com` → Your portfolio/website URL (if you have one)
- `YOUR_LINKEDIN` → Your LinkedIn username
- `YOUR_TWITTER` → Your Twitter handle (optional)

#### **Add Your Projects**
Replace the project placeholders with:
- Project names
- Descriptions
- Links to repositories
- Relevant tech badges

#### **Update About Me Section**
Customize the bio with:
- What you're currently working on
- Technologies you're learning
- Your interests and goals

### 3. Set Up Contribution Snake Animation

1. Copy the `README.md` and `.github/workflows/snake.yml` files to your profile repository
2. The GitHub Action will automatically run and generate the snake animation
3. The snake will update every 12 hours automatically
4. You can also manually trigger it from the "Actions" tab

### 4. Get Your GitHub Stats

The stats will automatically show up once you:
1. Replace `YOUR_GITHUB_USERNAME` with your actual username
2. Push the README to your profile repository
3. Wait a few minutes for the stats API to generate images

### 5. Customize Colors & Themes

Current theme is **Tokyo Night** with custom accent color `#D85A30` (orange-red).

To change colors:
- `title_color` → Header text color
- `icon_color` → Icon color
- `text_color` → Body text color
- `bg_color` → Background color

Available themes:
- `tokyonight` (current)
- `radical`
- `merko`
- `gruvbox`
- `dracula`
- `monokai`
- `dark`
- `nord`

### 6. Add More Tech Badges

Visit [shields.io](https://shields.io) or [simpleicons.org](https://simpleicons.org) to create custom badges.

**Badge format:**
```markdown
![Name](https://img.shields.io/badge/Name-HexColor?style=flat-square&logo=logoname&logoColor=white)
```

**Example:**
```markdown
![Rust](https://img.shields.io/badge/Rust-000000?style=flat-square&logo=rust&logoColor=white)
```

### 7. Customize Sections

Feel free to:
- Add or remove sections
- Change emojis (find more at [emojipedia.com](https://emojipedia.org))
- Modify table layouts
- Add images or GIFs
- Include additional stats or widgets

### 8. Optional Additions

**Profile View Counter:**
Already included at the bottom! Shows how many people visited your profile.

**WakaTime Stats:**
Track your coding time by adding:
```markdown
![WakaTime Stats](https://github-readme-stats.vercel.app/api/wakatime?username=YOUR_WAKATIME_USERNAME&theme=tokyonight)
```

**Trophy Stats:**
Add GitHub trophies:
```markdown
![Trophy](https://github-profile-trophy.vercel.app/?username=YOUR_GITHUB_USERNAME&theme=tokyonight&no-frame=true&row=1)
```

**Activity Graph:**
Add contribution graph:
```markdown
![Activity Graph](https://github-readme-activity-graph.vercel.app/graph?username=YOUR_GITHUB_USERNAME&theme=tokyo-night)
```

## 🚀 Deployment Steps

1. **Copy files** from this folder to your profile repository
2. **Commit and push** to the `main` branch
3. **Visit** your GitHub profile (`github.com/YOUR_USERNAME`)
4. **Your README will now display** on your profile!

## 🎯 Pro Tips

1. **Keep it updated** - Update projects and stats regularly
2. **Make it personal** - Add your own style and personality
3. **Use high-quality images** - If adding screenshots or photos
4. **Test locally** - Use a Markdown preview tool before pushing
5. **Be creative** - This is your digital business card!
6. **Pin your best repos** - Select 6 repositories to pin on your profile
7. **Add a profile picture** - Make sure you have a professional avatar
8. **Write good repo descriptions** - They show up on your profile
